## ğŸ”¹ What is a Function?

A **function** is a block of reusable code that performs a specific task.

* Instead of writing the same code again and again, you put it inside a function and **call** it whenever needed.
* Functions make code **modular, readable, and reusable**.

## ğŸ”¹ Parameters vs Arguments

* **Parameter** â†’ variable in function definition (`name` in `def greet(name)`)
* **Argument** â†’ actual value you pass (`"Harsh"` when calling `greet("Harsh")`)

---

## ğŸ”¹ Types of Function Arguments

1. **Positional Arguments**

   ```python
   def add(a, b):
       return a + b

   print(add(5, 3))  # 8
   ```

   (Order matters)

2. **Keyword Arguments**

   ```python
   print(add(b=3, a=5))  # 8
   ```

   (Order doesnâ€™t matter if you specify keywords)

3. **Default Arguments**

   ```python
   def greet(name="Guest"):
       return f"Hello, {name}"

   print(greet())        # Hello, Guest
   print(greet("Harsh")) # Hello, Harsh
   ```

4. **Variable-Length Arguments**

   * `*args` â†’ multiple positional arguments (tuple)
   * `**kwargs` â†’ multiple keyword arguments (dictionary)

   ```python
   def demo(*args, **kwargs):
       print("Positional:", args)
       print("Keyword:", kwargs)

   demo(1, 2, 3, name="Harsh", age=23)
   # Positional: (1, 2, 3)
   # Keyword: {'name': 'Harsh', 'age': 23}
   ```

## ğŸ”¹ Lambda (Anonymous) Functions

Used for **short one-line functions**:

```python

```

Often used with `map()`, `filter()`, `sorted()`.

## ğŸ”¹ Scope of Variables

* **Local variable** â†’ defined inside a function, accessible only there.
* **Global variable** â†’ defined outside all functions, accessible everywhere.

```python
x = 10  # global

def test():
    x = 5  # local
    print(x)

test()       # 5
print(x)     # 10
```
